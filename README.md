# Jules iOS App

Это мобильное приложение для Jules, написанное на SwiftUI.

## Как запустить проект

В данном репозитории находится исходный код приложения в формате Swift Package. Чтобы запустить его на телефоне или симуляторе, вам потребуется Xcode.

### Способ 1: Создание проекта в Xcode (Рекомендуемый)

Так как репозиторий содержит исходные файлы без файла проекта `.xcodeproj`, для сборки приложения выполните следующие шаги:

1. **Установите Xcode** из Mac App Store (требуется версия 15.0 или новее, так как проект использует iOS 17).
2. **Создайте новый проект**:
   - Откройте Xcode.
   - Выберите **"Create New Project..."**.
   - Выберите **iOS** -> **App**.
   - Нажмите **Next**.
   - Назовите проект `JulesApp`.
   - Organization Identifier: `com.yourname` (или используйте свой идентификатор).
   - Interface: **SwiftUI**.
   - Language: **Swift**.
   - Нажмите **Next** и сохраните проект в удобном месте.
3. **Добавьте файлы**:
   - В навигаторе проекта (слева) удалите автоматически созданные файлы `ContentView.swift` и `JulesAppApp.swift` (или аналогичный файл с `@main`).
   - Перетащите следующие папки и файлы из скачанного репозитория в папку вашего проекта в Xcode (в группу `JulesApp`):
     - `Models/`
     - `Network/`
     - `Utils/`
     - `ViewModels/`
     - `Views/`
     - `ContentView.swift`
     - `JulesApp.swift`
   - В появившемся окне убедитесь, что выбрана опция **"Copy items if needed"** и стоит галочка напротив таргета `JulesApp`.
4. **Настройка запуска на реальном устройстве**:
   - Подключите ваш iPhone к Mac кабелем.
   - В Xcode выберите ваш телефон в списке устройств (сверху, в центре панели инструментов).
   - Перейдите в настройки проекта (нажмите на корневой элемент `JulesApp` с синей иконкой в левом меню) -> выберите таргет `JulesApp` -> вкладка **Signing & Capabilities**.
   - В разделе **Signing** выберите вашу **Team** (Если списка нет, нажмите "Add Account..." и войдите с вашим Apple ID).
   - Убедитесь, что **Bundle Identifier** уникален (если требуется).
5. **Запуск**:
   - Нажмите кнопку **Play** (треугольник) в левом верхнем углу или сочетание клавиш `Cmd + R`.
   - Дождитесь сборки и установки приложения на устройство.

### Настройка API Key

Для работы приложения требуется API Key.
1. Запустите приложение на устройстве.
2. При первом запуске (или если ключ не введен) откроется экран настроек.
3. Введите ваш **Jules API Key**.

## Структура проекта

- **Models**: Модели данных приложения.
- **Network**: Сервисы для взаимодействия с API.
- **ViewModels**: View Models (MVVM).
- **Views**: SwiftUI представления (экраны и компоненты).
- **Utils**: Вспомогательные утилиты и тема оформления.
